This folder contains the file baldey.txt. It is described in: https://doi.org/10.1080/17470218.2014.984730

Furthermore, this folder contains data from CELEX for Dutch, which concern phonology (DPW.CD), morphology (DMW.CD, DML.CD), and frequency (DFW.CD, DFL.CD). The files can be downloaded from: http://catalog.elra.info/en-us/repository/browse/ELRA-L0029_01/
